# 4.0.3 - 2026-03-18

## Summary

This hotfix release fixes several bug in the new "generate-package-diffs" command.
Also adds a platform identifier to the tar gz file generated by the "export" command.
Furthermore, it fixes a bug related to the generation of the build-step package file during the "build" command.

## Bugs

 - #360: Fixed bugs in package diff generation
 - #363: Sorted channels field in build step package file
 - #365: Added platform name to exported tar gz file
